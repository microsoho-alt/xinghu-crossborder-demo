# 星狐AI跨境商品诊断舱 Demo v5+

这是外部演示增强版。

## v5升级内容

1. 增加三个演示案例下拉框：
   - 案例1：C端红海消费品｜车载手机支架
   - 案例2：B端工业品｜硬质合金铣刀
   - 案例3：带电小家电｜便携式迷你吸尘器

2. 诊断报告升级为可交付版：
   - 执行摘要
   - 产品基础画像
   - 经营测算模型
   - 七维评分矩阵
   - 平台适配与渠道建议
   - 风险提示与补强方向
   - OPC运营任务拆解
   - 7天小样本验证路径
   - 数据回流字段
   - 技术链路说明

3. 默认仍然支持手动填写真实产品数据。

## v5+ 增量能力

- 最多 4 张产品图或现场拍摄，预览、删除与 MIME/大小双端校验；
- 图片只生成待确认建议，不静默覆盖用户字段；
- 图片与文字汇入同一标准产品档案，再启动一次隐藏供应商细节的融合诊断；
- 可见进度、证据/推断/未知标签、分歧与置信度；
- 报告恢复码直接读取 PostgreSQL 快照，不重新运行分析；
- 实用性/准确性反馈与稍后实际结果回填。

原 v5 三个样本、全部输入、公式、七维评分、平台规则、风险、OPC 任务、7 天路径、数据回流和 Markdown 报告保持不变，并由黄金测试锁定。

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 公网部署

上传以下文件到 GitHub 仓库：

```text
app.py
requirements.txt
README.md
.streamlit/config.toml
```

